## 문제 8.

다음 코드 A와 B를 비교하시오.

```c
// A

while (i > 0) {
    printf("%d\n", i);
    --i;
}
```

```c
// B
int i = 0;

do {
    printf("%d\n", i);
    --i;
} while (i > 0);
```
